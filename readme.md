# 各高校镜像站前端信息统计

## 收集此信息的原因

网页前端是大多数初次使用镜像站的人会接触到的。设计相对合理的前端和性能良好的前端将有助于用户查找特定信息和展示当前状态。为方便后期继续接入 mirrorz 其他服务需要提前进行技术信息收集和准备。并检查自身易用性和传输性能。
